命令类型图例:
Motion (移动): 移动光标，或定义操作范围。
Command (命令): 直接执行动作。
Insert Mode Command (进入插入模式): 红色表示进入插入模式。
Operator (操作符): 需要配合一个移动命令，在光标和目标之间操作。
Extra (特殊): 特殊功能，通常需要额外输入。
重要提示:
  • :w 保存, :q 退出, :q! 不保存退出, :wq 保存并退出。
  • :e 文件名 打开文件。
  • :%s/旧/新/g 全局替换。
  • :h 获取帮助。
  • 数字 + 命令: 重复命令N次 (例如: 2p, d2w, 5j)。
  • dd 删除行, yy 复制行, cc 修改行。
  • gg 文件头, G 文件尾。
  • u 撤销, Ctrl-R 重做。
  • zt 光标行至屏幕顶, zb 底, zz 中。
